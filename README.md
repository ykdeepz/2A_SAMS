# Student Attendance System

A comprehensive student attendance management system built with Angular and JSON Server,
featuring complete data isolation between instructors and role-based access control.

---

## 🚀 Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Start servers (Windows)
start-dev.bat

# OR Manual start
npm run json-server  # Terminal 1
npm start            # Terminal 2
```

**Access**: http://localhost:4200  
**Login**: admin@school.com / admin123
